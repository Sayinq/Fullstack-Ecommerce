This is our NEXTJS full-stack project for a placeholder PC E-commerce website. 

TECH STACK:
NextJS
ReactJS (FRONT-END)
MongoDB
NodeJS (BACK-END)
Deploy with Netlify

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
